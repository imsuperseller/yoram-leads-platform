# Yoram Friedman Insurance - Lead Generation Platform

A Next.js landing page for Yoram Friedman Insurance Agency, designed to generate leads in the Israeli insurance market.

## Features

- **Hebrew RTL Support**: Fully optimized for Hebrew right-to-left layout
- **Mobile-First Design**: Responsive design optimized for mobile users
- **Lead Generation Form**: Simple 3-field form (name, phone, email)
- **WhatsApp Integration**: Direct WhatsApp contact button
- **SEO Optimized**: Meta tags and structured data for Hebrew search
- **Regulatory Compliant**: Meets Israeli insurance advertising requirements

## Tech Stack

- **Next.js 14** - React framework with TypeScript
- **Tailwind CSS** - Utility-first CSS framework
- **Heebo Font** - Google Fonts Hebrew typography

## Getting Started

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build

```bash
npm run build
npm start
```

## Key Components

- **Landing Page**: Optimized conversion page with trust indicators
- **Lead Form**: 3-field form following best practices
- **Mobile Optimization**: Touch-friendly design for Israeli mobile users
